Usage:
  gpt Meris.GlintCorrection [options]

Description:
  MERIS atmospheric correction using a neural net.


Source Options:
  -SaatsrProduct=<file>    The AATSR L1b input product.
                           This is an optional source.
  -SmerisProduct=<file>    The MERIS L1b input product.
                           This is a mandatory source.

Parameter Options:
  -PatmoNetFlintFile=<file>          The file of the atmospheric net to be used instead of the default neural net.
                                     Default value is '25x30x40_6936.3.net'.
  -PatmoNetMerisFile=<file>          The file of the atmospheric net to be used instead of the default neural net.
                                     Default value is '20x25x45_55990.1.net'.
  -PcloudIceExpression=<string>      The arithmetic expression used for cloud/ice detection.
                                     Default value is 'toa_reflec_14 > 0.2'.
                                     This is a mandatory parameter.
                                     Value must not be empty.
  -PderiveRwFromPath=<boolean>       Switch between computation of water leaving reflectance from path reflectance and direct use of neural net output
.
                                     Default value is 'false'.
  -PdoNormalization=<boolean>        Whether to perform normalization of bidirectional reflectances to nadir.
                                     Default value is 'false'.
  -PdoSmileCorrection=<boolean>      Whether to perform SMILE correction.
                                     Default value is 'false'.
  -PlandExpression=<string>          The arithmetic expression used for land detection.
                                     Default value is 'toa_reflec_10 > toa_reflec_6 AND toa_reflec_13 > 0.0475'.
                                     This is a mandatory parameter.
                                     Value must not be empty.
  -PoutputPath=<boolean>             Toggles the output of water leaving path reflectance.
                                     Default value is 'true'.
  -PoutputReflec=<boolean>           Toggles the output of water leaving irradiance reflectance.
                                     Default value is 'true'.
  -PoutputTosa=<boolean>             Toggles the output of TOSA reflectance.
                                     Default value is 'true'.
  -PoutputTransmittance=<boolean>    Toggles the output of downwelling irradiance transmittance.
                                     Default value is 'true'.
  -PuseFlint=<boolean>               Toggles the usage of a FLINT value in neural net.
                                     Default value is 'false'.
