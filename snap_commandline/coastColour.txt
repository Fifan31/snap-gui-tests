Usage:
  gpt CoastColour.L1P [options]


Source Options:
  -Sl1b=<file>    MERIS L1b (N1) product
                  This is a mandatory source.

Parameter Options:
  -Palgorithm=<cloudScreeningSelector>    Sets parameter 'algorithm' to <cloudScreeningSelector>.
                                          Value must be one of 'GlobAlbedo', 'QWG', 'CoastColour'.
                                          Default value is 'GlobAlbedo'.
  -PdoCalibration=<boolean>               Whether to perform the calibration.
                                          Default value is 'true'.
  -PdoEqualization=<boolean>              Perform removal of detector-to-detector systematic radiometric differences in MERIS L1b data products.
                                          Default value is 'true'.
  -PdoSmile=<boolean>                     Whether to perform SMILE correction.
                                          Default value is 'true'.
  -PuseIdepix=<boolean>                   Sets parameter 'useIdepix' to <boolean>.
                                          Default value is 'true'.

Usage:
  gpt CoastColour.L2R [options]


Source Options:
  -SsourceProduct=<file>    MERIS L1B or L1P product
                            This is a mandatory source.

Parameter Options:
  -Palgorithm=<cloudScreeningSelector>    Sets parameter 'algorithm' to <cloudScreeningSelector>.
                                          Value must be one of 'GlobAlbedo', 'QWG', 'CoastColour'.
                                          Default value is 'GlobAlbedo'.
  -PcloudIceExpression=<string>           The arithmetic expression used for cloud/ice detection.
                                          Default value is 'toa_reflec_14 > 0.2'.
                                          This is a mandatory parameter.
                                          Value must not be empty.
  -PdoCalibration=<boolean>               Whether to perform the calibration.
                                          Default value is 'true'.
  -PdoEqualization=<boolean>              Perform removal of detector-to-detector systematic radiometric differences in MERIS L1b data products.
                                          Default value is 'true'.
  -PdoSmile=<boolean>                     Whether to perform SMILE correction.
                                          Default value is 'true'.
  -PlandExpression=<string>               The arithmetic expression used for land detection.
                                          Default value is 'toa_reflec_10 > toa_reflec_6 AND toa_reflec_13 > 0.0475'.
                                          This is a mandatory parameter.
                                          Value must not be empty.
  -PuseIdepix=<boolean>                   Sets parameter 'useIdepix' to <boolean>.
                                          Default value is 'true'.

        <landExpression>string</landExpression>
        <cloudIceExpression>string</cloudIceExpression>
      </parameters>

Parameter Options:

  -PcloudIceExpression=<string>           The arithmetic expression used for cloud/ice detection.

  -PdoSmile=<boolean>                     Whether to perform SMILE correction.
