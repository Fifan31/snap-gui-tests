Usage:
  gpt CoastColour.L1P [options]

Description:
  Computes a refinement of top of atmosphere radiance and pixel characterization information.


Source Options:
  -SmerisL1B=<file>    The MERIS L1B input product
                       This is a mandatory source.

Parameter Options:
  -PccCloudBufferWidth=<int>                          The width of a cloud 'safety buffer' around a pixel which was classified as cloudy.
                                                      Valid interval is [0,100].
                                                      Default value is '2'.
  -PccCloudScreeningAmbiguous=<double>                Threshold of Cloud Probability Feature Value above which cloud is regarded as still ambiguous (i.e. a higher value results in fewer ambiguous clouds).
                                                      Default value is '1.4'.
  -PccCloudScreeningSure=<double>                     Threshold of Cloud Probability Feature Value above which cloud is regarded as sure (i.e. a higher value results in fewer sure clouds).
                                                      Default value is '1.8'.
  -PccIgnoreSeaIceClimatology=<boolean>               Check for sea/lake ice also outside Sea Ice Climatology area.
                                                      Default value is 'false'.
  -PccOutputCloudProbabilityFeatureValue=<boolean>    Write Cloud Probability Feature Value to the  CC L1P target product.
                                                      Default value is 'false'.
  -PdoCalibration=<boolean>                           Applies correction from MERIS 2nd to 3rd reprocessing quality.
                                                      Default value is 'false'.
  -PdoEqualization=<boolean>                          Perform removal of detector-to-detector systematic radiometric differences in MERIS L1b data products.
                                                      Default value is 'true'.
  -PdoIcol=<boolean>                                  Whether to perform ICOL correction (NOTE: This step can be very time- and memory-consuming! Please see help documentation for more details).
                                                      Default value is 'false'.
  -PdoSmile=<boolean>                                 Whether to perform MERIS Smile-effect correction.
                                                      Default value is 'true'.
